Antivirus App

About

A comprehensive program that combines a large number of different tools and utilities: antivirus, registry cleaner, vulnerability checker, anti-tamper functions, startup manager, etc.p manager, etc. The project was quite interesting in that it was necessary to combine a lot of different utilities into one interface, which were often interconnected. Particular attention has been paid to the scan and auto-correction screen: this is the main tool of the program. There, the errors were grouped into various subgroups, upon disclosure of which it was possible to get acquainted with each vulnerability found in more detail. Icons were drawn for each vulnerability and group. While working on the program, we constantly used interactive prototypes. In programs with so many different related functionality, this helped to effectively build connections between screens. A UI kit was also prepared, the entire interface was assembled taking into account the atomic approach. In the process, we used MS Windows libraries with some customization.
